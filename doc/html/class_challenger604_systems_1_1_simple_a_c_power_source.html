Detailed Description

Extends PowerSource with built-in fields for maximum, available, current, and requested power levels as well as voltage and AC frequency. This is useful for power sources that don't have to do complicated things with their power levels. This supplies AC electricity at nominal 115 volts/400 hertz. All these functions are virtual, so you can override them.
