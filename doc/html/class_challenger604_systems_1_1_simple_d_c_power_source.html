Detailed Description

Extends PowerSource with built-in fields for maximum, available, current, and requested power levels as well as voltage. This is useful for power sources that don't have to do complicated things with their power levels. This supplies DC electricity at nominal 28 volts. All these functions are virtual, so you can override them.
