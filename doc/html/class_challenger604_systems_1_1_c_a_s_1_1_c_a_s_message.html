Detailed Description

Base class for a Crew Alerting System message.

Every message that a system sends is one of these. A CAS message can have a priority level of warning, caution, advisory, or status.

As a result of the sending of a message, one or more of the following may happen:

  • A message may be displayed on the ECIAS screen
  • The aural warning system may sound an attention-getting sound and/or a voice warning
  • The master caution or master warning lights may illuminate

Member Enumeration Documentation

enum Challenger604Systems::CAS::CASMessage::Priority

An enumeration of priority levels for messages.

Enumerator:
WARNING 

A warning message (highest priority)

Warning messages are displayed in red. A warning message always triggers a triple attention-getting sound and turns on the master warning lights.

CAUTION 

A caution message (second highest priority)

Caution messages are displayed in yellow. A caution message always triggers a single attention-getting sound and turns on the master caution lights.

ADVISORY 

An advisory message.

STATUS 

A status message.
