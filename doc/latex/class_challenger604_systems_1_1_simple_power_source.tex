\subsection{Detailed Description}
Extends \hyperlink{class_challenger604_systems_1_1_power_source}{Power\-Source} with built-\/in fields for maximum, available, current, and requested power levels. This is useful for power sources that don't have to do complicated things with their power levels. All these functions are virtual, so you can override them.
